### Terminal to Editor

- **terminal emulator**: [windows terminal](https://github.com/microsoft/terminal)
- **shell**: zsh
  - **prompt**: [starship](https://github.com/starship/starship)
  - plugins
    - [wslu, ubuntu-wsl](https://wslutiliti.es/wslu/install.html)
    - default configuration: [oh-my-zsh](https://github.com/ohmybash/oh-my-bash)
    - search files [(fd+fzf)](https://mike.place/2017/fzf-fd/)
    - [zsh-completions](https://github.com/zsh-users/zsh-completions)
    - [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)
  - terminal/shell management: tmux
  - CLI AI: [zsh_codex](https://github.com/tom-doerr/zsh_codex) (alternative: [AI Shell](https://github.com/BuilderIO/ai-shell))
- **editor**: [LunarVim(NeoVim Distro)](https://github.com/LazyVim/LazyVim)
  - plugins
    - plugin manager: [lazy.nvim](https://github.com/folke/lazy.nvim)
